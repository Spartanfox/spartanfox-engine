# spartanfox-engine
  An libgdx game engine

## This engine works on these platforms:

* Android
* Desktop
* IOS
* Websites

## The source can be found within:

```
core\src\com\spartanfox\engine
```

## Compiling the game :

* Android
```
  gradlew.bat android:assemble
```

* Desktop
```
  gradlew.bat desktop:dist
```
